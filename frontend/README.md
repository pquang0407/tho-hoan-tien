# 🐰 Thỏ Hoàn Tiền (Rabbit Cashback)

Chào mừng đến với **Thỏ Hoàn Tiền** - Ứng dụng web giúp bạn chuyển đổi link mua sắm (Shopee, Lazada, TikTok Shop) thành link affiliate để nhận hoàn tiền lên đến 80% hoa hồng. Tất cả được gói gọn trong một giao diện thỏ siêu cute và dễ dùng!

## 🌟 Tính năng nổi bật
- **Giao diện thân thiện & Dễ thương:** UI/UX lấy cảm hứng từ bé thỏ ngộ nghĩnh, mang lại trải nghiệm vui vẻ, nhẹ nhàng khi mua sắm.
- **Chuyển đổi link "Thần Tốc":** Chỉ cần dán link sản phẩm gốc, hệ thống trả về link affiliate hợp lệ trong chớp mắt.
- **Theo dõi "Cà rốt" (Hoa hồng):** Dashboard minh bạch, cập nhật số dư, trạng thái đơn hàng và lịch sử rút tiền rõ ràng.
- **Đăng nhập siêu tốc:** Xác thực an toàn, nhanh gọn qua Google API hoặc OTP Email.

## 🛠️ Công nghệ sử dụng
- **Core:** React + Vite
- **Styling:** CSS/SCSS (hoặc Framework tùy chọn như Tailwind/Bootstrap)
- **State Management:** React Hooks (Context API)
